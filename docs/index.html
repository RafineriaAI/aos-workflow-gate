AOS Workflow Gate

A green PR can still rely on a control that is missing, stale, produced by the wrong app, or modified by the same PR.

AOS verifies the gate, not the code. It is a read-only pre-merge control assurance check for the exact commit, returning PASSWARNBLOCK with one reason, one next action, and replayable evidence.

Not another AI reviewer. Existing tools produce findings and statuses. AOS verifies whether the intended controls governed this exact change, including subject binding, producer identity, and verifier independence, then preserves the decision.
