# Self-Serve Activation Path

**Status: free self-serve advisory validation is open.** Guided onboarding,
paid pilots, Marketplace publication, and efficacy claims remain closed
while the [Hybrid Value Gate](../benchmarks/value/ASSESSMENT.md) is `NO_GO`.

The current path requires no mandatory contact and leaves the user with a
diagnosis and files that can be verified independently.

| Step | Surface | What you leave with |
| --- | --- | --- |
| Discover | Public repository, Pages, and exact-commit cases | A bounded claim and one working quickstart |
| Qualify | [Hybrid Value Gate](../benchmarks/value/README.md) | Separate signal, product-test, external-usability, field-utility, and publication states |
| Adopt | Self-Test Mode (no checkout, read-only) | A replayable decision record from your own pipeline in one PR |
| Understand | Job Summary: What AOS found / effect / Next | One dominant action and the exact files to keep |
| Verify | [TRUST.md](TRUST.md), offline `verify`, [case studies](case-studies/zero-required-checks.md) | Independent confirmation of the technical claim |
| Escalate (future) | Opens only after a `GO` decision | A bounded, written scoping record |

## Where a human is required

No human is required to install or evaluate the free advisory Action.
Feedback is optional and opt-in. A RafineriaAI human would enter only for
future pilot scoping and delivery ([GUIDED_PILOT.md](GUIDED_PILOT.md)),
after the required `GO` decision. The source remains Apache-2.0, with no
account, trial clock, or telemetry reporting that it was evaluated.

## Boundary

A future paid path would sell cross-repository decision governance and
evidence handling, not protection quality.
Nothing in it makes a repository secure or compliant, and decision records
remain `UNSIGNED_NOT_OFFICIAL`.
